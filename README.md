# Bspwm

Dotfiles for my Bspwm rice.

## Dependencies

- **Bspwm** (Window manager)
- **Sxhkd** (Shortcut daemon)
- **Polybar** (Panel)
- **Rofi** (Menu)

## Optional Dependencies

- **Alacritty** (Default terminal emulator)
- **Iosevka Nerd Font** (Default font)
- **Thunar** (Default file manager)
